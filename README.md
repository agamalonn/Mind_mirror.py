# MindMirror

MindMirror is a personalized emotional journaling assistant built with Python and OpenAI's GPT API. The project explores how conversational AI can support reflection, self-awareness, and structured journaling.

The focus is not to replace professional mental-health support, but to create a gentle writing companion that helps users organize thoughts, identify patterns, and reflect more intentionally.

## Project Focus

- AI-assisted journaling
- Personalized reflection prompts
- Conversational user experience
- Python integration with an LLM API
- Responsible framing for emotional-support tools

## Key Features

- Accepts free-form journal input from the user
- Uses GPT-based responses to reflect themes and emotions
- Encourages structured thinking and self-reflection
- Can be extended with mood tracking, summaries, or recurring insights
- Simple Python implementation suitable for experimentation

## Technical Skills Demonstrated

- Python application development
- OpenAI API integration
- Prompt design for a sensitive human-centered use case
- Product thinking around AI safety and user trust
- Translating an abstract idea into a working prototype

## Responsible Use Note

MindMirror is a journaling and reflection assistant. It is not a therapist, diagnosis tool, or emergency resource. Any production version of a tool like this should include privacy protections, clear user consent, secure handling of journal data, and crisis-support guidance.

## What I Learned

This project helped me understand how important prompt design and product boundaries are when working with AI. I practiced building an AI-powered experience where tone, safety, and user trust matter as much as the technical API call.

It also strengthened my interest in AI products that are useful, human-centered, and carefully scoped.
